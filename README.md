# AI Chatbot System

A customizable AI-powered chatbot system for organizations, built using RAG (Retrieval-Augmented Generation).

## Documentation
- [Implementation Plan](implementation_plan.md)
- [Task List](task.md)

## Tech Stack
- **Backend:** Python (FastAPI), LangChain, ChromaDB
- **Frontend:** React, Vite, Tailwind CSS
- **LLM:** Google Gemini API (or OpenAI)

## Features
- Custom data ingestion (PDFs, URLs)
- Embeddable chat widget (Script tag integration)
- Admin dashboard for bot management
